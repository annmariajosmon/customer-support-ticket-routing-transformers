# Customer Support Ticket Routing using Transformers

## Overview

A transformer-based NLP application that classifies customer support tickets into predefined categories using DistilBERT.

## Features

- Ticket classification
- Transformer-based NLP
- Streamlit interface
- Real-time prediction

## Tech Stack

- Python
- Transformers
- Hugging Face
- PyTorch
- Streamlit

## Run

```bash
streamlit run app.py
